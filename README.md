# CityListApp
 
## BE ##

Prerequisites
* JDK 17

### Running the application locally ###

* Clone repository CityList
* Maven clean install
* Run application


## FE ##

Prerequisites
* Install Node.js

### Running the application locally ###

* Clone repository CityList-FE
* In the command line, navigate to the local project location
* run npm i to install the node_modules
* run ng serve
